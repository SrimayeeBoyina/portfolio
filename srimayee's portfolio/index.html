Bubble Game

The Bubble Game is a JavaScript application where bubbles with randomly generated numbers appear on the screen for 60 seconds. The player's objective is to search and click on the bubbles with that numbers. Each correctly clicked bubble adds to the player's score. The challenge is to maximize the score within the given time limit, providing an engaging and time-sensitive gaming experience.

Tic-Tac-Toe

The Tic-Tac-Toe JavaScript application is a classic two-player game where participants take turns marking X and O on a 3x3 grid. The objective is to achieve a horizontal, vertical, or diagonal line of their symbol. The game employs HTML, CSS, and JavaScript to create an interactive and dynamic user experience, allowing players to enjoy the timeless challenge of Tic-Tac-Toe in a web-based format.
